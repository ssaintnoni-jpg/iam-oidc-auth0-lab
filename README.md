# IAM Hands-On Lab: Auth0 + OIDC

This project demonstrates a practical implementation of Identity and Access Management (IAM) using Auth0 and OpenID Connect (OIDC) Authorization Code Flow.

---

## What This Project Covers

- Identity Provider setup (Auth0)
- Application registration
- OIDC authentication flow
- Token exchange process
- JWT decoding and analysis

---

## Tools Used

- Auth0 (Identity Provider)
- OIDC Debugger
- JWT.io
- Postman

- OAuth 2.0 → Authorization
- OIDC → Authentication
- JWT → Secure identity token
- Client Secret → Application credential
